Why This Class
Standalone VR is the class that holds up under austere IT, variable bandwidth, and fast fielding — the conditions the upstream configuration implies.
The layering moment. When a second capability — say M18 pistol sustainment — is added later, the upstream cascade re-runs: new doctrine, new tasks, new environments, new gates, new pack. The hardware doesn't. Same headset, same compute, same kit. New capability, same boxes.
Option B — Editorial Spec Sheet. Magazine-style layout: big headlines, generous whitespace, field labels as small-caps eyebrow headers. Reads like a brief. Best for presentations, printouts, or customer-facing screenshots.
